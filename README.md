# Smart Restaurant Multi-Agent System (SRMAS)

## Overview
SRMAS is a compact multi-agent demonstration for small restaurants. It includes Billing, Inventory, and Analytics agents with a simple session & memory mechanism.

## Contents
- `notebook/Smart_Restaurant_Multi-Agent_System_SRMAS.ipynb` — main demo notebook
- `data/` — synthetic menu & inventory CSVs
- `README.md` — this file

## How to run
1. Open the notebook in Kaggle/Colab/local Jupyter.
2. Install dependencies: `pip install pandas matplotlib`
3. Run cells in order (Cell 1 → Cell 7).
4. See outputs and demo evaluations.
